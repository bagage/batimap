État du bâti

Cette carte affiche l'état du bâti dans OpenStreetMap, par rapport au cadastre français. Il permet aussi de mettre à jour le bâti pour une commune donnée, via l'éditeur JOSM.

Cette opération comporte des risques et demande une expérience avancée sur la contribution à OpenStreetMap, consultez tout d'abord le wiki.

Pour faciliter la mise à jour du bâti avec JOSM, via le plugin Conflation, l'utilisation de scripts, via le plugin Scripting, permet d'éviter les tâches répétitives :

  1. A-segmented.js : crée une todolist, via le plugin Todo, des bâtiments segmentés nécessitant une revue humaine avant intégration dans OSM.
  2. B-conflation.js : configure le plugin Conflation.
